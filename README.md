## 项目简介

#### 本项目是用python+django搭建的一个校园二手市场信息发布平台。前端还用了bootstrap技.

## 项目运行方式

**确保你的开发环境是 python3，如果不是，请考虑使用虚拟环境virtualenv搭建python3, 并安装最新版Django,自行度娘并参照相关教程。**

   1.fork 本项目到你的仓库

2. 克隆你的仓库到本地

3. 在manage.py所在目录执行

   ```
   python manage.py runserver
   ```

 4.浏览器输入：http://127.0.0.1:8000/
